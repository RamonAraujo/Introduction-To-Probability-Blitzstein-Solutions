4.3.7 problem 45

Notice that

I(A1∩A2⋯∩An)≥n∑i=1I(Ai)−n+1

because the left-hand side is either 0, or 1, so the question reduces to whether the left-hand side is ever 0, while the right-hand side is 1. Notice that this is not possible, becuase if the left-hand side is 0, then Aj=0 for some j. Thus, ∑ni=1I(Ai)<n⟹R.H.S.<1.

Then,

I(n⋂i=1Ai)≥n∑i=1I(Ai)−n+1⟹E(I(n⋂i=1Ai))≥E(n∑i=1I(Ai)−n+1)⟹P(n⋂i=1Ai)≥n∑i=1P(Ai)−n+1
