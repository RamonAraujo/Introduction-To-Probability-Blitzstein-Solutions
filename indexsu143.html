4.1.10 problem 10

The probability that the game lasts n rounds is 1∕2n.

Thus, if the winnings for n rounds is n, we must compute ∑∞i=1(i∕2i).

We know that ∑∞i=1(xi)=x1−x. Deriving both sides with respect to x gives ∑∞i=1(ixi−1)=1(1−x)2. Multiplying by both sides gives ∑∞i=1(ixi)=x(1−x)2. Plugging in x=1∕2 gives the answer 2.

For the second part of the problem we need to find ∑∞i=1(i2∕2i).

We know ∑∞i=1(ixi)=x(1−x)2. Deriving both sides with respect to x again, using the quotient rule, gives ∑∞i=1(i2xi−1)=1+x(1−x)3. Multiplying both sides by x gives ∑∞i=1(i2xi)=x+x2(1−x)3. Plugging in x=1∕2 gives the answer of 6.
